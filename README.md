# Terminal Portfolio Site

A terminal-themed personal portfolio built with vanilla HTML, CSS, and JavaScript using AI's help.

Live: [LucasAnselmo-git.github.io](https://LucasAnselmo-git.github.io)

## Features

- Interactive terminal with command-based navigation
- Scanline CRT visual effect
- Easter Eggs
- Fully responsive for mobile devices

## Security Measures

- XSS sanitization on all user input and dynamic content
- URL protocol validation to prevent javascript: injection
- Reverse tabnabbing protection 
- Content Security Policy header
- Input rate limiting to prevent spam
- Command input length restriction

## Tech Stack

HTML, CSS, JavaScript

## Commands

- `help` — directory of available commands
- `about` — background info
- `projects` — project list with links
- `contact` — contact info and links
- `clear` — clears terminal
